## Overview

This catalog contains 190 meticulously crafted themes designed for enterprise-grade applications across all platforms. Each theme uses a standardized 6-token color system that ensures perfect compatibility with shadcn/ui components and seamless migration between platforms.

### Token System

Every theme includes 6 essential color tokens:

- **primary**: Main brand color for primary actions
- **accent**: Secondary color for highlights and interactive elements
- **background**: Base background color
- **surface**: Card/container background color
- **text**: Primary text color
- **border**: Border and divider color

## shadcn/ui Integration

All 190 themes are designed to work seamlessly with shadcn/ui components. The 6-token system maps directly to CSS custom properties:

```css
:root {
  --primary: theme.tokens.primary;
  --accent: theme.tokens.accent;
  --background: theme.tokens.background;
  --surface: theme.tokens.surface;
  --text: theme.tokens.text;
  --border: theme.tokens.border;
}
```

### Component Mapping

| Token | shadcn/ui Usage |
|-------|----------------|
| `primary` | Buttons, primary actions, focus states |
| `accent` | Links, highlights, secondary actions |
| `background` | Page background, body |
| `surface` | Cards, modals, popovers |
| `text` | Body text, headings |
| `border` | Borders, dividers, outlines |

## Installation & Deployment

### Single-Click Installation

All themes can be installed with a single API call:

```bash
curl -X POST http://your-domain.com/api/themes/sync \
  -H "Content-Type: application/json" \
  -d '{"deviceId":"my-device","themeId":"nebula"}'
```

### Cross-Device Sync

Apply the same theme across all your devices:

```javascript
// Web
await fetch('/api/themes/sync', {
  method: 'POST',
  body: JSON.stringify({
    deviceId: 'user-123',
    themeId: 'tokyo-nights'
  })
});

// Mobile (React Native)
const response = await fetch('https://api.your-domain.com/api/themes/sync', {
  method: 'POST',
  body: JSON.stringify({
    deviceId: deviceInfo.id,
    themeId: 'tokyo-nights'
  })
});

// Desktop (Electron)
ipcRenderer.send('sync-theme', {
  deviceId: machineId,
  themeId: 'tokyo-nights'
});
```

### Theme Migration

The standardized 6-token structure allows seamless migration:

```javascript
// Export from one platform
const currentTheme = {
  primary: '#ec4899',
  accent: '#06b6d4',
  background: '#0f172a',
  surface: '#1e293b',
  text: '#fce7f3',
  border: '#f472b6'
};

// Import to another platform (identical structure)
applyTheme(currentTheme);
```

## Best Practices

1. **Contrast Ratios**: All themes maintain WCAG AA contrast ratios (4.5:1 minimum)
2. **Color Blindness**: Test critical themes with color blindness simulators
3. **Platform Consistency**: Use the same theme ID across all platforms for brand consistency
4. **User Preference**: Store user theme preference and sync across devices
5. **Dynamic Switching**: Implement smooth transitions when switching themes
6. **Default Fallback**: Always have a default theme (e.g., `daylight` or `midnight`)
